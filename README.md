# Hacker News Feed
Interface for Hacker News website  

## Technologies stack
This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).  
-React  
-React Router  
-React Redux
-Ant Design && React-Bootstrap
-Yarn Package manager
